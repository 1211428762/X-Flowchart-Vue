# X-Flowchart

> A flowchart editor plugin based on vue.

| :-------- |

## Example

``` bash
# git clone https://github.com/OXOYO/X-Flowchart-Vue.git

# install dependencies
yarn install

### Compiles and hot-reloads for development
yarn run serve
```
